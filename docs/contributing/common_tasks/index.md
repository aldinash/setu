# Common Development Tasks

This section provides step-by-step guides for common development tasks in Setu. These guides are designed to help new contributors get started quickly and ensure consistency across the codebase.


## Contributing Guidelines

When working on any of these tasks, please follow our:
- [Development Workflow](../development_workflow.md)
- [C++ Style Guide](../style_guides/cpp_style_guide.md)
- [Python Style Guide](../style_guides/python_style_guide.md)

## Getting Help

If you encounter issues while following these guides:
1. Check the [troubleshooting documentation](../../troubleshooting.md)
2. Search existing [GitHub issues](https://github.com/setu-project/setu/issues)
3. Ask questions in our [discussions forum](https://github.com/setu-project/setu/discussions)
